# WebAppTest

## 개요
프로필 이미지와 프로필 정보를 보여주는 간단한 웹앱.
초기 개발부터 **확장성**을 우선한다 — 프로필에 새 기능/항목이 계속 추가될 수 있음.
스택: Next.js (App Router) + TypeScript + CSS Modules.

## 작업 규칙 (사전 규칙)
- 모호한 요구사항이나 추가 질문 사항이 있으면 **작업 시작 전에 먼저 물어본다.**

## 문서 관리 규칙
- 개발하면서 갱신되는 모든 상세 문서는 `Documents/` 폴더에서 관리한다.
- 이 CLAUDE.md 에는 개요 + 사전 규칙 + 아래 문서 목록만 둔다. 상세 내용은 적지 않는다.

### Documents 목록
- `Documents/SETUP.md` — 로컬 실행 / 테스트 방법
- `Documents/ARCHITECTURE.md` — 폴더 구조 + 확장성 설계 (새 기능 추가 방법)
- `Documents/EDITING.md` — 프로필 이미지 / 정보 편집 방법 (코드 수정 불필요)
